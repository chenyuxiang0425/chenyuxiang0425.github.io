静态方法和非静态方法

概念

静态方法（static methods)，也可称为类方法（class methods）；

非静态方法 (non-static methods)，也可称为实例方法 (instance methods)。

非静态方法只能被该 class 的特定实例调用；

静态方法可以被 class 本身调用（可以被实例调用，但不建议这么做）；

this关键词则不能用于静态方法。

Python 爬虫爬取网页中表格的数据

Python 爬虫爬取网页中表格的数据

问题描述

做硕士课题的时候需要获得分子描述符，网上找了找发现有相应数据库，但同一个分子的分子描述符被拆分成好几个网页。因此需要爬取网页中各个表格的数据，整合到一个表上，再写入 Excel 文件。

以 http://michem.disat.unimib.it/mole_db/desc_values.php?id=31690&desc_group=2 为例，需要爬取网页中表格的数据，和desc_group=1，2，4，5.....等中数据的表格。

基本思路

  1. requests 包中 get 方法请求网站，得到一个 HTTPresponse 类型；
  2. bs4 解析库解析网页中的源代码，定位表格；
  3. 数据以 [['Name', 'Value']] 的方式储存；
  4. 用pandas库写入 Excel 文件。

代码实现

为何在 Java我们可用 System.out.println(5 + "horse")，而在 Python不能用 print(5 + "horse")？

发表于 2019-12-05 更新于 2020-08-18 分类于 Java ， CS61B

原因

考虑这两个 Java 语句

和

第一个能成功运行，但第二个有类型错误。Java 是强类型的。如果认为 h 是 string，它可以连接两个元素并且给你一个字符串。 但如果 h 是 int，Java 不能连接一个数字和一个字符串，如果给你一个字符串。

Python 没有约束类型，它不会对你想要的类型做出假设。对于 x = 5 + "horse"到底应该是字符串还是数字，Python 不知道，因此它报错了。

这个情况下，System.out.println(5 + "horse");，Java将参数解释为字符串，并输出 5horse 。更有用的， System.out.println(5 + " ");，5 后面加个空格。
