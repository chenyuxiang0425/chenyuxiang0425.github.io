Objects

  • 对象代表着信息
  • 它包含数据和行为，捆绑在一起以创建抽象
  • 对象可以代表事物，但也可以代表属性、交互和过程
  • 对象的类型称为类;类是 Python 中的头等值
  • 面向对象的程序设计
    • 组织大型程序的比喻
    • 可以改进程序组成的特殊语法
  • Python 中的每个值都是对象
    • 所有对象都有属性
    • 许多数据的处理是通过对象方法
    • 函数只做一件事情，但对象做多个事情(做每个事情都是一个函数)
