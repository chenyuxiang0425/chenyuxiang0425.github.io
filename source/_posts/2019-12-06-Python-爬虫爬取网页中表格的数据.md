# Python 爬虫爬取网页中表格的数据

## 问题描述

做硕士课题的时候需要获得分子描述符，网上找了找发现有相应数据库，但同一个分子的分子描述符被拆分成好几个网页。因此需要爬取网页中各个表格的数据，整合到一个表上，再写入 Excel 文件。

以 http://michem.disat.unimib.it/mole_db/desc_values.php?id=31690&desc_group=2 为例，需要爬取网页中表格的数据，和`desc_group=1，2，4，5.....`等中数据的表格。



## 基本思路

1.  `requests` 包中 get 方法请求网站，得到一个 HTTPresponse 类型；
2. ` bs4`  解析库解析网页中的源代码，定位表格；
3.  数据以 `[['Name', 'Value']]`  的方式储存；
4.  用`pandas`库写入 Excel 文件。



## 代码实现

```python
import requests
from bs4 import BeautifulSoup
import pandas as pd


# 爬虫[Requests 设置请求头 Headers],伪造浏览器
headers = {
    'User-Agent':'Mozilla/5.0 (Windows NT 6.1; WOW64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/63.0.3239.132 Safari/537.36'
}  

# 指定爬虫网页
url = 'http://michem.disat.unimib.it/mole_db/desc_values.php?id=31690&desc_group='
url_list = []
for num in [1,2,4,5,6,8, 10, 12, 15, 16, 17, 18, 20]:
    url_list.append(url + str(num))

# 获取网页中表格表格
listData=[['Name', 'Value']] 
for url in url_list:
    params = {"show_ram": 1}
    response = requests.get(url, params=params, headers=headers)  # 访问url
    soup = BeautifulSoup(response.text, 'html.parser')            # 获取网页源代码
    tr = soup.find('table').find_all('tr')  # .find 定位到所需数据位置 .find_all 查找所有的tr（表格）
	# 写入 listData
    for j in tr[1:]:                         # tr2[1:] 遍历第1列到最后一列，表头为第0列
        td = j.find_all('td')                # td 表格
        Name = td[1].get_text().strip()      # 遍历 Name
        Value = td[2].get_text().strip()     # 遍历 Value
        listData.append([Name, Value])

# listData 写入 Excel
data = pd.DataFrame(listData)
writer = pd.ExcelWriter('MDEA.xlsx')		          # 写入 Excel
data.to_excel(writer, 'page_1', float_format='%.5f')  # ‘page_1’是写入excel的sheet名
writer.save()
writer.close()
```
