# 静态方法和非静态方法



## 概念

静态方法（static methods)，也可称为类方法（class methods）；

非静态方法 (non-static methods)，也可称为实例方法 (instance methods)。

非静态方法只能被该 class 的特定实例调用；

静态方法可以被 class 本身调用（可以被实例调用，但不建议这么做）；

`this`关键词则不能用于静态方法。



## 用静态方法的原因

`Math` class 提供的一个 `sqrt` 方法，因为它是静态方法，所有我们这么调用：

```java
x = Math.sqrt(100);
```

如果 `sqrt` 是实例方法，我们只能用这种尴尬的调用方式：

```java
Math m = new Math();
x = m.sqrt(100);
```



## 详解

### 背景

对于 `Dog` 这个 class，要比较两只狗的体重。

```java
public class Dog {
    public int weightInPounds;
    
    public Dog(int w){
        weightInPounds = w;
    }
}
```



### 静态方法调用

同一个 class 可以同时有静态方法和非静态方法，例如，我们要比较两只狗的体重，可以实现的一种方式是增加一个比较 Dogs 的静态方法。

```java
public static Dog maxDog(Dog d1, Dog d2) {
    if (d1.weightInPounds > d2.weightInPounds) {
        return d1;
    }
    return d2;
}
```

这个方法以这种方式被调用：

```java
Dog d1 = new Dog(15);
Dog d2 = new Dog(100);
Dog.maxDog(d1, d2);
```



### 非静态方法调用

当然也可用非静态方法实现这个操作：

```java
public Dog maxDog(Dog d2) {
    if (this.weightInPounds > d2.weightInPounds){
        return this;
    }
    return d2;
}
```

我们可以用 `this` 关键词指代当前 object。这个方法可以这么被调用：

```java
Dog d1 = new Dog(15);
Dog d2 = new Dog(100);
d1.maxDog(d2);
```



## 额外思考锻炼

**Exercise 1.2.1**: What would the following method do? If you're not sure, try it out.

```java
public static Dog maxDog(Dog d1, Dog d2) {
    if (weightInPounds > d2.weightInPounds) {
        return this;
    }
    return d2;
}
```



引用自 [CS61B 1.2 Class Methods vs. Instance Methods](https://joshhug.gitbooks.io/hug61b/content/chap1/chap12.html)。











### 答案

这里会有两个错误：

1. `this`关键词则不能用于静态方法；
2. `weightInPounds`是非静态变量，不能由静态方法调用，想被调用必须先实例化。
