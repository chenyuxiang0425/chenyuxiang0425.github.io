  • forEach 中的 function 最多可以用三个参数，依次含义为：
    • foreach被调用的数组中的每个元素(每次循环迭代)。
    • 第二个表示该元素的索引。
    • 第三个表示foreach被调用的数组(对于循环的每次迭代都是相同的)。
