sum

sum(iterable, start=0) -> value

  • 返回“开始”值(默认值:0)加上一个可迭代的数字，当可迭代为空时，返回开始值。

max

max(…)
max(iterable, , key=func) -> value
max(arg1, arg2, *args, *[, key=func]) -> value

all & any

all(iterable, /)
如果bool(x)对迭代中的所有x都为真，则返回真。
如果可迭代为空，则返回True。
any(iterable, /)
如果bool(x)对迭代中的有一个x为真，则返回真。
如果可迭代为空，则返回False。
